# 美光科技 - 投资笔记

## 核心投资逻辑

### 1. AI存储超级周期核心受益者

**AI驱动需求**：
- AI服务器对HBM需求激增，全球HBM市场规模以40%年复合增长率从2025年350亿美元增长至2028年1000亿美元
- AI服务器单台DRAM需求是普通服务器的8倍
- HBM供不应求，仅能满足客户50%-67%需求，短缺持续至2026年后
- 2026全年HBM长约锁死

**业绩爆发**：
- 2026财年Q2营收238.6亿美元（+196.4%）
- 净利润137.85亿美元（+770.8%）
- 毛利率74.4%，接近历史峰值

### 2. HBM突破

**HBM进展**：
- HBM3E已成功应用于英伟达H200 TensorCore GPU
- HBM4已批量出货（英伟达定制），2026年Q2高良率量产
- HBM4E 2027年量产，目标带宽超2.8TB/s
- HBM市占30%+，增速超三星

**技术优势**：
- HBM3E带宽高达1.2TB/s，引脚速度超9.2Gb/s
- 性能较主要竞争对手高出约10%
- 功耗节省约30%

### 3. 寡头垄断格局

**行业壁垒**：
- DRAM/NAND CR3≈95%，行业壁垒极高
- 技术、产能、资金壁垒森严
- 头部厂商充分享受周期红利

**美光地位**：
- 全球第三大存储芯片巨头
- 与三星、SK海力士形成三足鼎立
- 美股唯一大型存储芯片IDM厂商

### 4. 先进制程领先

**1γ DRAM**：
- 采用EUV极紫外光刻技术
- 位密度较1β提升30%以上
- 2026年中成主力制程

**G9 NAND**：
- 支持高达3.6 GB/s的I/O传输速率
- 2026年中量产

### 5. 绑定AI巨头

**客户结构**：
- 深度合作英伟达、AMD、云厂商
- 2025财年前十大客户收入贡献超总收入一半
- 单一最大客户占比从2024财年10%提升至2025财年17%
- 长单锁定增长

## 财务亮点

### 业绩爆发
- 2025财年营收373.78亿美元（+48.85%）
- 2025财年净利润85.39亿美元（+997.56%）
- 2026财年Q2毛利率74.4%，接近历史峰值
- 2026财年Q2指引：营收183-191亿美元，毛利率67%-69%

### 业务结构优化
- 云存储业务2025财年增速256.6%
- 云存储+核心数据中心合计贡献55.5%营收
- 增长引擎已完成从"移动端消费"向"AI云基建"的根本性切换

### 现金流改善
- 2025财年Q4自由现金流19.5亿美元
- 现金及等价物122.2亿美元
- 资金状况稳健

## 风险提示

### 1. 周期性风险
- 存储行业强周期性，毛利率从-9%到74%
- 接近历史峰值，涨价空间有限
- 强周期见顶风险

### 2. 资本开支风险
- 未来2年超500亿投入
- 折旧侵蚀利润
- 需求不及预期将巨亏
- 美国本土投资近2000亿美元，回报周期长

### 3. 竞争风险
- 三星、SK海力士扩产HBM
- 三星30年来一直选择市场份额而非利润率
- 国产长鑫/长江存储追赶成熟制程
- HBM市场份额落后于三星和SK海力士

### 4. 地缘政治风险
- 美国对华限制影响中国市场
- 供应链依赖韩国设备

### 5. 需求单一依赖风险
- AI服务器占比过高
- 投资降温将引发需求断崖
- 单一最大客户占比17%

- 密切关注三星价格策略和HBM扩产进度

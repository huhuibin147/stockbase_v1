## 主营业务

### 核心产品

1. **固态硬盘类产品（SSD）**（收入占比42.46%，45.82亿元）
   - 消费级SSD和企业级SSD
   - 企业级SSD于2025年第三季度开始量产
   - 受益于AI服务器、数据中心需求爆发
   - QLC NAND技术商用，产品结构持续升级
   - 业务毛利率：28.65%

2. **嵌入式存储类产品**（收入占比33.95%，36.63亿元）
   - eMMC、UFS（嵌入式多芯片封装）
   - LPDDR（低功耗双倍数据速率）
   - 产品具备高耐久、宽温域特性，适配高价值场景
   - 广泛应用于手机、车载电子、AI终端
   - 业务毛利率：32.18%（公司盈利水平最高的核心板块）

3. **移动存储类产品**（收入占比13.66%，14.74亿元）
   - 存储卡、存储盘（U盘）
   - 自研移动存储主控芯片
   - 主要聚焦移动存储市场

4. **内存条类产品**（收入占比9.74%，10.51亿元）
   - DDR3、DDR4、DDR5系列规格
   - RDIMM、SODIMM、UDIMM、LPCAMM2、CAMM2等类型
   - 受益于AI服务器需求

### 核心技术能力

- **闪存主控芯片自研**：公司核心壁垒，国内少数同时掌握"前端晶圆资源整合、存储控制器芯片研发+固件研发、模组制造、渠道销售"全链路能力的存储企业
- **自研主控芯片型号**：TW8581、TW2983、TW8381、TW2981、TW2980等
- **固件算法优势**：LDPC纠错+磨损均衡算法强，能把低端颗粒"救活"
- **技术差异化**：更适配国产CPU平台，与三星等国际巨头竞争时有差异化优势

### 供应链合作

- **上游晶圆厂**：与海力士（SK hynix）、闪迪（SanDisk）、英特尔（Intel）等存储原厂的主要经销商建立长期合作关系
- **国产替代**：2020年11月成为长江存储（YMTC）Xtacking 3D NAND金牌生态合作伙伴，开始批量采购和产品验证
- **国产DRAM**：直接受益于长鑫存储国产DRAM量产（8000Mbps DDR5，良率80%）

## 商业模式

### 盈利模式
- **产品销售**：存储模组产品直销（SSD、嵌入式存储、内存条、移动存储）
- **技术赋能**：自主可控的主控芯片与固件方案，提供一站式、全链路存储解决方案
- **定制化服务**：从底层技术到终端场景的全链路布局，场景化、定制化解决方案

### 定价策略
- 消费级产品：性价比优势，抢占中低端市场
- 企业级产品：高附加值，定制化解决方案溢价
- 嵌入式存储：技术壁垒高，溢价显著

### 供应链特点
- **全链路能力**：国内少数掌握"主控芯片+固件方案+存储模组"全链路格局的企业
- **激进备货策略**：2025年末存货70.58亿元（占总资产65%），充分享受涨价红利
- **库存是"门票"**：存储模组企业打入大客户供应链需先向上游锁定晶圆产能，预付款从2025年初0.8亿涨到年末2.96亿

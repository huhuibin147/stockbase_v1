# 德明利 - 投资笔记

## 投资逻辑

### 核心投资逻辑

**AI驱动的存储超级周期 + 激进备货策略 = 业绩爆发式增长**

1. **行业β**：AI大模型训练/推理驱动存储芯片需求爆发，全球存储价格持续飙涨
2. **公司α**：
   - 2025年行业低谷期前瞻性备货70.58亿，充分享受涨价红利
   - 自研主控芯片能力，技术差异化优势（适配国产CPU平台）
   - 企业级SSD于2025年Q3开始量产，切入高附加值市场
3. **业绩弹性**：2026年Q1单季净利润31.5-36.5亿，相当于2025年全年的5倍

### 投资亮点

✅ **业绩爆发确定性强**
- 2026年Q1净利润31.5-36.5亿，同比+4659%-5383%
- 库存超100亿，在涨价周期将持续受益
- AI服务器需求爆发，企业级存储订单激增

✅ **存储超级周期前中段**
- 机构共识：当前存储处于AI超级景气周期前中段
- 2026、2027年，甚至更长时间里供求紧张关系不能改变
- 不管技术路线如何变化，存储需求是无差异的

✅ **国产替代受益**
- 长江存储金牌生态合作伙伴
- 直接受益长鑫存储国产DRAM量产
- 国家政策大力支持半导体国产化

✅ **产品结构优化**
- 主动收缩低毛利消费级存储业务
- 重点布局AI服务器大容量企业级SSD
- 高毛利的内存条和嵌入式存储业务占比提升

✅ **毛利率持续改善**
- 2025年Q4毛利率已提升至27.14%
- 预计2026年毛利率将持续上行
- 嵌入式存储毛利率32.18%，SSD毛利率28.65%

### 投资风险

⚠️ **存货减值风险（核心风险）**
- 2025年末存货70.58亿，占总资产65%
- 存货跌价准备计提仅0.5%（同行平均2.3%）
- 如果2026年下半年存储价格涨不动或下跌，70亿库存将变成巨额亏损
- 计提减值可能导致业绩大幅波动，股价暴跌

⚠️ **资金链风险**
- 资产负债率69.87%，显著高于行业平均
- 经营现金流-22.41亿，严重"失血"
- 囤货资金全部借来，如果货卖不掉或回款慢，资金链会非常紧张
- 财务费用2.02亿，利息负担重

⚠️ **周期反转风险**
- 存储行业强周期性，当前处于周期高位
- 历史经验：周期顶部往往伴随业绩爆发，但反转也很快
- 需密切关注存储价格走势和供需关系

⚠️ **技术短板**
- 高端UFS/旗舰级暂无突破（相比江波龙UFS4.1+5nm）
- 专利质量和前沿性不如江波龙
- 研发团队规模较小（350+人 vs 江波龙近1000人）

⚠️ **估值过高**
- 当前股价496元，总市值约1125亿
- 静态PE 163倍（基于2025年净利润）
- 如果2026年业绩不达预期，估值将面临大幅回调

### 操作建议

#### 买入策略

**条件**：
1. 存储价格持续上涨（跟踪DRAMeXchange/NAND Flash价格指数）
2. 公司2026年Q2业绩继续高增长（验证Q1不是昙花一现）
3. 股价回调至400-450元区间（更好的安全边际）

**建仓**：
- 当前位置（496元）：轻仓试探（10-20%仓位）
- 回调至450元：加仓（20-30%仓位）
- 回调至400元：重仓（30-40%仓位）

#### 持有策略

**继续持有的条件**：
1. AI服务器需求持续旺盛
2. 存储价格保持上涨或高位稳定
3. 公司季度业绩持续高增长
4. 经营现金流改善（回款正常）

#### 卖出策略

**减仓条件**（触发任意一条减仓30-50%）：
1. 存储价格开始下跌（连续2个月环比下跌）
2. 公司季度业绩增速放缓（环比下降）
3. 存货跌价准备计提比例大幅提升（从0.5%提升至2%+）
4. 资产负债率继续上升（突破75%）

**清仓条件**（触发任意一条清仓）：
1. 存储价格暴跌（单月跌幅超20%）
2. 公司季度业绩环比大幅下滑（降幅超30%）
3. 出现重大财务风险（债务违约、资金链断裂）
4. 技术面破位（股价跌破200元，跌破长期支撑）

### 仓位控制

**建议仓位**：不超过总仓位的20%（高风险高收益标的）

**理由**：
- 业绩弹性极强，但风险也极高
- 存货减值可能导致业绩大幅波动
- 资金链紧张，存在不确定性

## 主营业务

### 核心产品

1. **AI智算集群**（收入占比约70.75%）
   - 基于自研GPU的大规模智算集群解决方案
   - 支持万亿参数大模型训练
   - 千卡GPU智算集群效率超过同等规模国外同代系GPU集群

2. **GPU板卡**（收入占比约17.74%）
   - MTT S5000：旗舰级训推一体全功能GPU智算卡
   - MTT S4000/S3000：数据中心与AI计算卡
   - MTT S80/S70：消费级显卡

3. **一体机**（收入占比约9.05%）
   - 集成GPU计算的一体化解决方案
   - 面向企业级客户

4. **其他产品**（收入占比约2.41%）
   - 芯片及其他相关产品

### 重要客户

- **智算中心**：各地政府智算中心项目
- **互联网企业**：头部互联网公司
- **企业客户**：需要进行AI训练和推理的大型企业
- **政务客户**：政府部门的数字化转型项目

### 应用领域

| 应用领域 | 占比 | 说明 |
|----------|------|------|
| AI智算 | 主要 | 大模型训练与推理 |
| 数字孪生 | 快速增长 | 城市、工业数字孪生 |
| 图形渲染 | 次要 | 专业图形加速 |
| 科学计算 | 次要 | 高性能计算 |
| 消费电子 | 次要 | 游戏显卡等 |

## 商业模式

### 盈利模式
- **产品销售**：GPU芯片、板卡、集群系统直销
- **解决方案**：提供AI算力一体化解决方案
- **服务收入**：技术支持、运维服务

### 定价策略
- 相比国际同类产品具有性价比优势
- 通过国产替代概念获取政策支持
- 针对不同客户群体提供差异化产品

### 供应链特点
- **Fabless模式**：专注于芯片设计，制造外包
- 晶圆制造：交由台积电等晶圆代工厂
- 封装测试：交由专业封测厂商
- 板卡加工：交由代工厂生产

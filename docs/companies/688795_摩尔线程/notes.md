# 摩尔线程 - 投资笔记

## 一、投资逻辑

### 1.1 核心投资逻辑

**国产GPU替代+AI算力爆发双轮驱动**

1. **国产替代刚性需求**
   - 美国对华GPU出口管制持续升级
   - 国内GPU国产化率仅15%，替代空间巨大
   - 政策大力支持半导体自主可控

2. **AI算力需求爆发**
   - 大模型训练需要海量算力
   - 2024年全球GPU市场规模1.05万亿元，预计2029年增至3.61万亿元
   - 五年复合增长率超27%

3. **全功能GPU稀缺性**
   - 国内少数同时具备图形渲染+AI计算能力的GPU企业
   - 差异化竞争，避开与华为昇腾等在AI训练芯片的正面竞争
   - 应用场景更广泛（AI+图形+数字孪生）

### 1.2 成长驱动因素

1. **产品迭代**
   - 四代GPU架构快速迭代
   - MTT S5000性能接近英伟达H100的61%
   - 下一代产品规划中

2. **生态建设**
   - MUSA架构兼容CUDA生态
   - 降低开发者迁移成本
   - 2025年首届MUSA开发者大会

3. **客户拓展**
   - 从政府智算中心向企业客户延伸
   - 互联网大厂合作深化
   - 消费级市场逐步开拓

4. **产能释放**
   - Fabless模式，产能依赖代工厂
   - 随着产品放量，规模效应显现

### 1.3 潜在风险

1. **技术风险**
   - 与英伟达差距仍然巨大（H100是2022年产品）
   - GPU技术迭代极快，追赶难度大
   - 生态建设需要长期投入

2. **竞争风险**
   - 国际：英伟达、AMD垄断（国内市场份额超90%）
   - 国内：华为昇腾、寒武纪、海光信息、沐曦等
   - 市场份额不足1%

3. **财务风险**
   - 持续亏损（累计未弥补亏损超16亿元）
   - 高研发投入（年13亿元+）
   - 客户极度集中（前五大客户占比98%）

4. **估值风险**
   - 市值3000亿元，市销率约200倍
   - 已大幅透支未来3-5年预期
   - 若业绩不及预期，股价回调风险大

5. **供应链风险**
   - Fabless模式依赖台积电等代工厂
   - 地缘政治可能影响晶圆代工

## 二、估值分析

### 2.1 估值方法

**市销率（PS）估值法**（适用于尚未盈利的高成长科技公司）

| 可比公司 | 2025年营收(亿元) | 市值(亿元) | PS |
|---------|----------------|-----------|-----|
| 摩尔线程 | 15.06 | ~3000 | ~200x |
| 寒武纪 | 64.97 | ~5700 | ~88x |
| 海光信息 | 143.76 | ~4900 | ~34x |

**分析**：
- 摩尔线程PS远高于可比公司
- 反映市场对其高成长性和稀缺性的溢价
- 但也意味着高估值风险

### 2.2 合理估值区间

**乐观情景**（2027年盈利）：
- 假设2027年营收50亿元，净利润5亿元
- 给予100倍PE（高成长科技股）
- 合理市值：500亿元
- 当前市值3000亿元，**明显高估**

**中性情景**（2028年盈利）：
- 假设2028年营收80亿元，净利润10亿元
- 给予60倍PE
- 合理市值：600亿元
- 当前市值3000亿元，**明显高估**

**悲观情景**（持续亏损）：
- 若2027年仍未盈利
- 按PS 50倍（寒武纪水平）
- 假设2027年营收50亿元
- 合理市值：2500亿元
- 当前市值3000亿元，**略高估**

### 2.3 估值结论

**当前估值偏高，已透支未来3-5年成长预期**

- 合理市值区间：500-1000亿元
- 当前市值：~3000亿元
- **高估幅度：200%-500%**

## 三、投资观点

### 3.1 长期看好逻辑

1. **赛道好**
   - GPU是AI时代的核心基础设施
   - 国产替代是确定性趋势
   - 市场空间巨大（万亿级别）

2. **团队强**
   - 创始人张建中是英伟达前全球副总裁
   - 核心团队来自国际GPU巨头
   - 技术+商业双重能力

3. **进展快**
   - 6年完成四代GPU架构迭代
   - 2025年营收15亿元，同比增243%
   - 2026Q1首次盈利

4. **生态建设**
   - MUSA兼容CUDA，降低迁移门槛
   - 专利514项，国内领先

### 3.2 短期担忧

1. **估值过高**
   - 上市首日涨425%，市值冲上3000亿
   - 中签者单签盈利24万
   - 透支未来业绩

2. **业绩波动**
   - 虽然2026Q1盈利，但扣非仍亏损0.54亿
   - 盈利可持续性待观察

3. **理财争议**
   - 募资80亿，计划用75亿理财
   - 市场质疑资金使用效率

4. **解禁压力**
   - 2026年下半年面临首批解禁
   - 早期投资者可能减持

### 3.3 投资建议

**短期（6个月内）：观望**
- 当前估值过高，不宜追高
- 等待估值回调至合理区间（1000亿以下）
- 或等待业绩进一步验证

**中期（1-2年）：逢低布局**
- 若市值回调至1000-1500亿，可考虑建仓
- 关注季度业绩是否持续改善
- 关注新产品发布和生态进展

**长期（3-5年）：坚定持有**
- 若能在市值合理时买入
- 持有至公司实现稳定盈利
- 享受国产GPU替代红利

### 3.4 操作策略

**建仓条件**：
1. 市值回落至1000-1500亿区间
2. 连续2个季度扣非净利润为正
3. 经营现金流转正
4. 新产品发布且性能大幅提升

**加仓条件**：
1. 年度营收增速保持50%以上
2. 毛利率稳定在65%以上
3. 客户结构多元化（前五大客户<70%）
4. MUSA生态显著进展

**减仓条件**：
1. 营收增速降至30%以下
2. 毛利率跌破50%
3. 研发进展不及预期
4. 核心技术人员流失

## 五、与绿的谐波对比

### 5.1 相似之处

- 都是**国产替代**核心标的
- 都处于**高研发投入期**
- 都受益于**产业趋势**（机器人/GPU）
- 都是**科创板**上市公司

### 5.2 差异之处

| 维度 | 摩尔线程 | 绿的谐波 |
|------|---------|---------|
| 行业 | GPU芯片 | 机器人零部件 |
| 盈利状态 | 2026Q1首次盈利 | 2025年已盈利 |
| 市值 | ~3000亿 | ~200亿 |
| PS | ~200倍 | ~30倍 |
| 竞争格局 | 英伟达垄断 | 哈默纳科主导 |
| 国产化率 | 15% | 30%+ |
| 确定性 | 较低（技术风险大） | 较高（已验证） |

### 5.3 投资选择

- **确定性**：绿的谐波 > 摩尔线程
- **成长空间**：摩尔线程 > 绿的谐波
- **估值合理性**：绿的谐波 > 摩尔线程
- **风险收益比**：绿的谐波更优

## 六、总结

### 6.1 一句话评价

**"好赛道、好团队、好进展，但价格太贵"**

### 6.2 投资纪律

1. **不追高**：当前估值透支未来，耐心等待回调
2. **看业绩**：连续2-3个季度盈利验证后再考虑
3. **控仓位**：即使买入，仓位不超过5%
4. **设止损**：若买入后跌20%，坚决止损
5. **长思维**：以3-5年视角看待，不被短期波动影响

### 6.3 最终建议

**暂时观望，等待更好买点**

- 目标买入市值：800-1200亿元
- 对应股价：约170-255元（按4.7亿股计算）
- 当前股价：约638元
- **需回调60-73%**

若市场出现系统性调整，或公司业绩不及预期导致估值回落，将是较好的布局时机。

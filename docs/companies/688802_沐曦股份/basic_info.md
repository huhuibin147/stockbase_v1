## 主营业务

### 核心产品线

1. **曦云C系列（训推一体GPU）**（收入占比超70%）
   - **曦云C500/C550**：主力产品，2024年2月量产
     - 采用自研XCORE 1.0架构
     - 搭载64GB HBM2e显存
     - FP16/BF16算力处于英伟达A100区间
     - 支持2卡至64卡多种互连拓扑
   - **曦云C600**：2025年WAIC发布，预计2025年底风险量产
     - 性能介于A100和H100之间
     - 支持FP8等多精度混合算力
     - 全流程国产供应链闭环
   - **曦云C700**：研发中，预计2026下半年流片
     - 综合性能对标英伟达H100
     - 基于国产供应链打造

2. **曦思N系列（AI推理GPU）**（收入占比约20%）
   - **曦思N100**：2023年4月量产
     - 单卡算力160TOPS（INT8）、80TFLOPS（FP16）
     - 最高128路编码和96路解码视频处理能力
     - 面向智算推理场景

3. **曦彩G系列（图形渲染GPU）**（研发中）
   - 聚焦专业图形渲染赛道
   - 预计2026-2027年推出

4. **曦索X系列（科学智能）**（研发中）
   - 深耕科学智能领域
   - 面向科研、工程计算

## 商业模式

### 盈利模式
- **芯片销售**：GPU板卡、加速卡直销
- **计算平台**：提供GPU计算平台解决方案
- **软件授权**：软件栈、开发工具配套

### 定价策略
- 相比英伟达同类产品具有性价比优势
- 通过国产替代获取政策支持
- 针对不同场景提供差异化产品

### 供应链特点
- **Fabless模式**：专注芯片设计，制造外包
- **先进封装**：支持2.5D CoWoS-L和Silicon Bridge等封装形态
- **国产化推进**：曦云C600实现从设计到封测全流程国产供应链闭环
- **灵活封装**：可根据需求将单颗或多颗SoC芯片与HBM芯片进行封装集成

# 沐曦股份 - 投资笔记

## 一、投资逻辑

### 1.1 核心投资逻辑

**国产AI芯片替代+AMD团队量产能力双轮驱动**

1. **国产替代刚性需求**
   - 美国对华AI芯片出口管制持续升级
   - 国内AI芯片国产化率仅15%，替代空间巨大
   - 政策大力支持半导体自主可控

2. **AI算力需求爆发**
   - 大模型训练需要海量算力
   - 2024年全球GPU市场规模1.05万亿元，预计2029年增至3.61万亿元
   - 五年复合增长率超27%

3. **AMD团队稀缺性**
   - 核心创始团队来自AMD，近20年GPU研发经验
   - 主导过十多款主流GPU产品，量产经验丰富
   - 芯片从研发到量产周期短

### 1.2 成长驱动因素

1. **产品迭代**
   - 曦云C500（对标A100）已量产
   - 曦云C600（介于A100-H100）2025年底风险量产
   - 曦云C700（对标H100）2026下半年流片

2. **生态建设**
   - MXMACA软件栈兼容CUDA生态
   - 端到端兼容PyTorch、TensorFlow等主流框架
   - 降低客户迁移成本

3. **客户拓展**
   - 国家级智算中心部署
   - 服务器厂商合作（新华三、超讯通信）
   - 在手订单14.30亿元

4. **产能释放**
   - Fabless模式，产能依赖代工厂
   - C600实现全流程国产供应链闭环
   - 先进封装能力（2.5D CoWoS-L）

### 1.3 潜在风险

1. **技术风险**
   - 与英伟达差距仍然巨大（C500对标A100，A100是2020年产品）
   - GPU技术迭代极快，追赶难度大
   - 生态建设需要长期投入

2. **竞争风险**
   - 国际：英伟达、AMD垄断（国内市场份额超90%）
   - 国内：华为昇腾、寒武纪、摩尔线程、海光信息等
   - 市场份额仅约1%

3. **财务风险**
   - 持续亏损（累计亏损超32亿元）
   - 高研发投入
   - 应收账款7.26亿，占营收44%

4. **估值风险**
   - 市值2800亿元，市销率约170倍
   - 已大幅透支未来3-5年预期
   - 若业绩不及预期，股价回调风险大

5. **供应链风险**
   - Fabless模式依赖代工厂
   - 地缘政治可能影响晶圆代工（C600实现国产闭环是亮点）

## 二、估值分析

### 2.1 估值方法

**市销率（PS）估值法**（适用于尚未盈利的高成长科技公司）

| 可比公司 | 2025年营收(亿元) | 市值(亿元) | PS |
|---------|----------------|-----------|-----|
| 沐曦股份 | 16.44 | ~2800 | ~170x |
| 摩尔线程 | 15.06 | ~3000 | ~200x |
| 寒武纪 | 64.97 | ~5700 | ~88x |
| 海光信息 | 143.76 | ~4900 | ~34x |

**分析**：
- 沐曦股份PS低于摩尔线程，但高于寒武纪、海光
- 反映市场对其高成长性的溢价
- 但也意味着高估值风险

### 2.2 合理估值区间

**乐观情景**（2026年盈利）：
- 假设2026年营收30亿元，净利润3亿元
- 给予80倍PE（高成长科技股）
- 合理市值：240亿元
- 当前市值2800亿元，**明显高估**

**中性情景**（2027年盈利）：
- 假设2027年营收50亿元，净利润5亿元
- 给予60倍PE
- 合理市值：300亿元
- 当前市值2800亿元，**明显高估**

**悲观情景**（持续亏损）：
- 若2027年仍未盈利
- 按PS 50倍（寒武纪水平）
- 假设2027年营收50亿元
- 合理市值：2500亿元
- 当前市值2800亿元，**略高估**

### 2.3 估值结论

**当前估值偏高，已透支未来3-5年成长预期**

- 合理市值区间：300-800亿元
- 当前市值：~2800亿元
- **高估幅度：250%-830%**

## 三、投资观点

### 3.1 长期看好逻辑

1. **赛道好**
   - AI芯片是AI时代的核心基础设施
   - 国产替代是确定性趋势
   - 市场空间巨大（万亿级别）

2. **团队强**
   - 创始团队来自AMD，技术背景深厚
   - 主导过十多款主流GPU产品
   - 量产经验丰富，研发到量产周期短

3. **进展快**
   - 5年完成四大产品线布局
   - 2025年营收16.44亿，同比增121%
   - 亏损收窄44%

4. **生态建设**
   - MXMACA兼容CUDA，降低迁移门槛
   - 医疗影像等垂直领域模型迁移基本无需调整

### 3.2 短期担忧

1. **估值过高**
   - 上市首日暴涨755%，市值突破3000亿
   - 发行价104.66元，当前股价718元
   - 透支未来业绩

2. **业绩波动**
   - 2025年仍亏损7.89亿
   - 2026Q1预告仍亏损9,076万-18,151万

3. **应收账款**
   - 应收账款7.26亿，占营收44%
   - 需关注回款风险

4. **解禁压力**
   - 2026年下半年面临首批解禁
   - 早期投资者可能减持

### 3.3 投资建议

**短期（6个月内）：观望**
- 当前估值过高，不宜追高
- 等待估值回调至合理区间（800亿以下）
- 或等待业绩进一步验证

**中期（1-2年）：逢低布局**
- 若市值回调至500-800亿，可考虑建仓
- 关注季度业绩是否持续改善
- 关注C600、C700量产进展

**长期（3-5年）：坚定持有**
- 若能在市值合理时买入
- 持有至公司实现稳定盈利
- 享受国产AI芯片替代红利

### 3.4 操作策略

**建仓条件**：
1. 市值回落至500-800亿区间
2. 连续2个季度扣非净利润为正
3. 经营现金流转正
4. C600成功量产并交付

**加仓条件**：
1. 年度营收增速保持50%以上
2. 毛利率稳定在55%以上
3. 客户结构多元化
4. C700流片成功

**减仓条件**：
1. 营收增速降至30%以下
2. 毛利率跌破45%
3. 研发进展不及预期
4. 核心技术人员流失

## 五、与摩尔线程对比

### 5.1 相似之处

- 都是**国产GPU替代**核心标的
- 都处于**高研发投入期**
- 都受益于**AI算力爆发**产业趋势
- 都是**科创板**上市公司（2025年12月上市）
- 都**尚未盈利**，纳入科创成长层

### 5.2 差异之处

| 维度 | 沐曦股份 | 摩尔线程 |
|------|---------|---------|
| 团队背景 | AMD背景（陈维良等） | 英伟达背景（张建中） |
| 产品定位 | AI芯片为主（C系列、N系列） | 全功能GPU（图形+AI） |
| 主力产品 | 曦云C500（对标A100） | MTT S5000（接近H100的61%） |
| 2025年营收 | 16.44亿 | 15.06亿 |
| 2025年净利润 | -7.89亿 | -10.01亿 |
| 毛利率 | 56.5% | 65.6% |
| 市值 | ~2800亿 | ~3000亿 |
| PS | ~170倍 | ~200倍 |
| 国产化 | C600实现国产闭环 | 依赖台积电 |
| 生态 | MXMACA兼容CUDA | MUSA兼容CUDA |

### 5.3 投资选择

- **盈利能力**：沐曦股份 > 摩尔线程（亏损更小）
- **毛利率**：摩尔线程 > 沐曦股份（65.6% vs 56.5%）
- **国产化**：沐曦股份 > 摩尔线程（C600国产闭环）
- **全功能**：摩尔线程 > 沐曦股份（图形+AI）
- **估值合理性**：沐曦股份 > 摩尔线程（PS 170倍 vs 200倍）
- **风险收益比**：沐曦股份略优

## 六、总结

### 6.1 一句话评价

**"好赛道、好团队、好进展，但价格太贵"**

### 6.2 投资纪律

1. **不追高**：当前估值透支未来，耐心等待回调
2. **看业绩**：连续2-3个季度盈利验证后再考虑
3. **控仓位**：即使买入，仓位不超过5%
4. **设止损**：若买入后跌20%，坚决止损
5. **长思维**：以3-5年视角看待，不被短期波动影响

### 6.3 最终建议

**暂时观望，等待更好买点**

- 目标买入市值：400-600亿元
- 对应股价：约100-150元（按4亿股计算）
- 当前股价：约718元
- **需回调79-86%**

若市场出现系统性调整，或公司业绩不及预期导致估值回落，将是较好的布局时机。

**与摩尔线程比较**：
- 两者估值都偏高，沐曦股份略优（PS 170倍 vs 200倍）
- 沐曦股份亏损更小，国产化进展更好
- 摩尔线程全功能GPU稀缺性更强
- 建议优先关注沐曦股份，等待合理估值

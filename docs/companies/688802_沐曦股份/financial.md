## 八、财务风险提示

### 8.1 主要风险

1. **持续亏损风险**
   - 预计2026年才可能盈亏平衡
   - 若营收增速放缓，亏损可能持续

2. **高研发投入风险**
   - 年研发投入持续增长
   - 技术迭代快，需持续高投入

3. **客户集中风险**
   - 主要客户为智算中心、服务器厂商
   - 客户集中度较高

4. **估值风险**
   - 市销率约170倍
   - 股价已大幅透支未来预期

5. **应收账款风险**
   - 应收账款7.26亿，占营收44%
   - 需关注回款风险

### 8.2 风险缓解因素

- IPO募资39亿，资金储备充足
- 营收高速增长，规模效应显现
- 亏损持续收窄，盈利拐点临近
- 国产替代政策持续支持

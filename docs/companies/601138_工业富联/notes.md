# 工业富联 - 投资笔记

## 一、投资逻辑

### 1.1 核心投资逻辑

**AI算力基础设施"卖水人"+全球AI服务器代工龙头**

1. **AI算力需求爆发**
   - 全球大模型竞赛催生算力饥渴
   - 2025年八大云厂商资本开支总额年增65%
   - AI服务器需求持续爆发

2. **全球AI服务器代工龙头**
   - 全球AI服务器市占率超40%
   - 英伟达GB200、GB300系列主力代工厂
   - 深度绑定英伟达，技术壁垒高

3. **业绩确定性高**
   - 订单能见度延伸至2027年
   - 2025年营收9029亿（+48%），净利润353亿（+52%）
   - 2026年延续高增长（预计营收+45-55%）

### 1.2 成长驱动因素

1. **AI服务器放量**
   - 英伟达GB200系列量产爬坡
   - AI服务器营收同比增长超3倍
   - Q4同比增幅超5.5倍

2. **产品升级**
   - 从通用服务器到AI服务器
   - 系统级机柜批量交付（行业首个）
   - 液冷散热等关键技术

3. **客户拓展**
   - 亚马逊、微软、谷歌等全球头部客户
   - 云服务商服务器营收同比增长超1.8倍

4. **产能扩张**
   - 全球化产能布局
   - 自动化产线全面跑通
   - 交付效率和成本控制能力质的飞跃

### 1.3 潜在风险

1. **客户集中风险**
   - 北美云厂商占比较大
   - 若客户削减资本开支，影响较大

2. **地缘政治风险**
   - 中美科技博弈
   - 供应链可能受影响

3. **竞争风险**
   - 广达、纬创、英业达等台湾同行
   - 浪潮信息等国内厂商

4. **毛利率风险**
   - 代工模式毛利率较低（10.5%）
   - 价格战可能压缩利润空间

## 二、估值分析

### 2.1 估值方法

**市盈率（PE）估值法**（已盈利公司）

| 可比公司 | 2025年净利润(亿元) | 市值(亿元) | PE |
|---------|-------------------|-----------|-----|
| 工业富联 | 352.9 | ~10800 | ~30x |
| 浪潮信息 | 25 | ~600 | ~24x |
| 中科曙光 | 12 | ~500 | ~42x |

**分析**：
- PE 30倍，处于合理区间
- 低于中科曙光（42倍），高于浪潮信息（24倍）
- 考虑到增速（52%）和行业地位，估值合理

### 2.2 合理估值区间

**乐观情景**（高增长持续）：
- 假设2026年净利润500亿元（+42%）
- 给予35倍PE（高成长）
- 合理市值：17500亿元
- 当前市值10800亿元，**低估**

**中性情景**（增速放缓）：
- 假设2026年净利润450亿元（+28%）
- 给予30倍PE
- 合理市值：13500亿元
- 当前市值10800亿元，**略低估**

**悲观情景**（增速大幅下降）：
- 假设2026年净利润380亿元（+8%）
- 给予25倍PE
- 合理市值：9500亿元
- 当前市值10800亿元，**略高估**

### 2.3 估值结论

**当前估值合理，略有低估**

- 合理市值区间：13000-15000亿元
- 当前市值：~10800亿元
- **低估幅度：17-28%**
- 当前股价：约54元
- 合理股价：约65-75元

## 三、投资观点

   - 全球云厂商资本开支持续加码
   - 市场空间巨大（万亿级别）

   - 深度绑定英伟达（核心合作伙伴）
   - 技术壁垒高（液冷、高密组装等）

3. **业绩确定**
   - 订单能见度延伸至2027年
   - 2025年净利润353亿，+52%
   - 2026年预计延续高增长

   - 业绩增速52%，PEG<1

   - 单一客户风险

3. **毛利率低**
   - 代工模式毛利率仅10.5%
   - 利润空间有限

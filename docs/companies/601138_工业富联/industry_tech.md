# 工业富联 - 行业与技术分析

## 一、AI服务器行业概览

### 1.1 行业定义

**AI服务器**
- 搭载高性能GPU的服务器
- 用于AI模型训练和推理
- 是AI算力基础设施的核心

**与传统服务器的区别**：
- **硬件**：搭载GPU（英伟达H100、GB200等）
- **性能**：算力是传统服务器数十倍
- **价格**：单价是传统服务器数倍至数十倍
- **毛利率**：高于传统服务器

### 1.2 行业分类

| 类型 | 特点 | 代表产品 | 主要厂商 |
|------|------|---------|---------|
| 训练服务器 | 大模型训练 | HGX H100、GB200 | 工业富联、广达 |
| 推理服务器 | AI推理 | L40S、L20 | 工业富联、纬创 |
| 通用服务器 | 传统计算 | PowerEdge、ProLiant | 戴尔、HPE |

### 1.3 市场规模

**全球市场**：
- 2024年：约1500亿美元
- 2025年（预计）：约2500亿美元
- 2027年（预计）：约4000亿美元
- 三年CAGR：约40%+

**驱动力**：
- 全球云厂商资本开支持续加码
- 2025年八大云厂商资本开支总额年增65%
- 大模型竞赛催生算力需求

### 1.4 行业驱动力

1. **AI大模型爆发**
   - GPT-4等万亿参数模型需要海量算力
   - 训练一次大模型需数千张GPU
   - 推理需求持续增长

2. **云厂商资本开支**
   - 亚马逊、微软、谷歌、Meta等
   - 2025年资本开支总额年增65%
   - 主要投向AI服务器和AI数据中心

3. **企业AI化**
   - 各行各业引入AI
   - 私有化部署需求
   - 边缘计算需求

4. **国产替代**
   - 国内智算中心建设
   - 国产AI芯片+服务器

## 二、竞争格局

### 2.1 全球AI服务器代工格局

**寡头竞争**：
- **工业富联**：全球市占率超40%，龙头地位
- **广达电脑**：全球市占率约20%，紧随其后
- **纬创资通**：全球市占率约10%
- **英业达**：全球市占率约8%
- **超微电脑（Supermicro）**：全球市占率约7%
- **其他**：约15%

**行业壁垒**：
- **制造规模**：大规模量产能力
- **客户认证**：云厂商认证周期长
- **供应链整合**：上游零部件供应链
- **技术能力**：液冷、高密组装等

### 2.2 中国AI服务器格局

| 公司 | 定位 | 优势 | 劣势 | 市值(亿) |
|------|------|------|------|---------|
| **工业富联** | 代工制造 | 全球龙头、英伟达绑定 | 毛利率低 | ~10800 |
| **浪潮信息** | 自主品牌 | 国内市占率超45% | 毛利率低 | ~600 |
| **中科曙光** | 自主品牌 | 自主可控、液冷技术 | 规模小 | ~500 |
| **联想集团** | 自主品牌 | 全球PC龙头 | AI服务器占比低 | - |

### 2.3 工业富联竞争地位

**优势**：
1. **全球龙头**
   - 全球AI服务器市占率超40%
   - 规模效应显著
   - 成本优势

2. **深度绑定英伟达**
   - GB200、GB300系列主力代工厂
   - 英伟达A100、H100板卡独家供应商
   - 英伟达HGX/DGX系列芯片基板独家供应（市场份额超50%）

3. **客户资源**
   - 亚马逊AWS、微软Azure、谷歌云等
   - 全球头部云厂商
   - 订单能见度延伸至2027年

4. **技术能力**
   - 液冷散热技术
   - 高密组装技术
   - 800G光模块+GPU集群服务器批量生产
   - 系统级机柜批量交付（行业首个）

**劣势**：
1. **代工模式**
   - 毛利率较低（10.5%）
   - 议价能力相对弱

2. **客户集中**
   - 北美云厂商占比较大
   - 单一客户风险

3. **地缘风险**
   - 中美科技博弈
   - 供应链可能受影响

## 三、技术分析

### 3.1 AI服务器技术壁垒

**硬件技术**：
1. **高密组装**
   - 多GPU集成（8卡、16卡）
   - 高密度布线
   - 散热设计

2. **液冷散热**
   - 冷板式液冷
   - 浸没式液冷
   - 散热效率提升30%+

3. **高速互联**
   - 800G光模块
   - GPU集群服务器
   - 低延迟、高带宽

**软件技术**：
1. **系统集成**
   - 服务器+网络+存储
   - 系统级机柜
   - 整体解决方案

2. **智能制造**
   - 自动化产线
   - 质量控制
   - 交付效率

### 3.2 工业富联技术路线

**核心技术**：
1. **液冷技术**
   - 冷板式液冷机柜
   - 系统级液冷解决方案
   - 行业领先

2. **高密组装**
   - 多GPU服务器组装
   - 自动化产线
   - 调试周期大幅缩短

3. **系统级机柜**
   - 行业内首个实现批量交付
   - 服务器+网络+存储+液冷
   - 整体交付

4. **800G光模块**
   - 高速网络
   - GPU集群互联
   - 批量生产能力

**产品矩阵**：

| 产品 | 定位 | 特点 | 客户 |
|------|------|------|------|
| GB200 AI服务器 | 训练 | 英伟达主力代工 | 云厂商 |
| GB300 AI服务器 | 训练 | 新一代产品 | 云厂商 |
| HGX系列 | 训练/推理 | 芯片基板独家供应 | 英伟达 |
| DGX系列 | 训练 | 芯片基板独家供应 | 英伟达 |
| 液冷机柜 | 散热 | 系统级批量交付 | 云厂商 |
| 800G交换机 | 网络 | 全球份额超75% | 云厂商 |

### 3.3 技术趋势

1. **液冷普及**
   - AI服务器功耗提升
   - 风冷无法满足散热需求
   - 液冷成为标配

2. **系统级交付**
   - 从单台服务器到系统级机柜
   - 整体解决方案
   - 提升附加值

3. **高速网络**
   - 800G光模块普及
   - 1.6T光模块研发中
   - GPU集群互联需求

4. **智能化制造**
   - 自动化产线
   - 质量控制
   - 交付效率

## 四、产业链分析

### 4.1 上游供应链

**GPU芯片**：
- 主要供应商：英伟达
- 产品：H100、GB200、GB300等
- **风险**：地缘政治可能影响

**零部件**：
- CPU：Intel、AMD
- 内存：三星、海力士、美光
- 存储：三星、西数等
- 光模块：中际旭创、新易盛等

### 4.2 中游制造

**工业富联（JDM模式）**：
- 联合设计制造
- 从设计到制造一站式服务
- 深度绑定客户

**核心竞争力**：
- 规模制造能力
- 技术能力（液冷、高密组装）
- 交付效率
- 成本控制

### 4.3 下游应用

**主要客户类型**：
1. **云服务巨头**（主要）
   - 亚马逊AWS
   - 微软Azure
   - 谷歌云
   - Meta等

2. **企业客户**
   - 大型企业内部AI
   - 私有化部署

3. **智算中心**
   - 政府智算中心
   - 商业化智算中心

### 4.4 产业链风险

1. **供应链风险**
   - GPU依赖英伟达
   - 地缘政治影响

2. **客户集中风险**
   - 北美云厂商占比较大
   - 单一客户风险

3. **政策风险**
   - 出口管制
   - 关税政策

## 五、行业趋势

### 5.1 技术趋势

1. **GPU迭代加速**
   - 英伟达每年推出新产品
   - 性能持续提升
   - 服务器需要适配

2. **液冷普及**
   - AI服务器功耗提升
   - 液冷成为标配
   - 市场空间大

3. **系统级交付**
   - 从单台到系统级机柜
   - 整体解决方案
   - 提升附加值

### 5.2 市场趋势

1. **云厂商资本开支持续加码**
   - 2025年+65%
   - 2026年预计继续增长
   - 主要投向AI服务器

2. **AI服务器渗透率提升**
   - 从训练到推理
   - 从云端到边缘
   - 从大厂到中小企业

3. **行业集中度提升**
   - 龙头优势明显
   - 小厂商被淘汰
   - 工业富联市占率提升

### 5.3 政策趋势

1. **支持政策**
   - 新基建
   - 智算中心建设
   - 产业政策

2. **出口管制**
   - 美国对华AI芯片管制
   - 影响国内客户
   - 但海外客户不受影响

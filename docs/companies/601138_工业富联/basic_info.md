## 主营业务

### 三大核心业务板块

1. **云计算业务**（收入占比约66.7%）
   - **AI服务器**：核心增长引擎
     - 英伟达GB200、GB300系列AI服务器主力代工厂
     - 全球AI服务器市占率超40%
     - 2025年AI服务器营收同比增长超3倍
   - **通用服务器**：传统服务器制造
   - **液冷机柜**：系统级机柜批量交付
   - **存储设备**：数据中心存储解决方案
   
2. **通信及移动网络设备**（收入占比约25%）
   - **高速交换机**：全球份额超75%
     - 800G交换机进入量产阶段
     - 客户涵盖思科、华为、亚马逊等
   - **路由器**：企业级、运营商级
   - **5G设备**：5G基站及相关设备
   - **通信网络设备高精密机构件**

3. **工业互联网**（收入占比约8%）
   - **灯塔工厂解决方案**：对外赋能
     - 对内打造8座灯塔工厂
     - 对外赋能9座世界级灯塔工厂
   - **工业互联网平台**：连续6年入选工信部双跨平台
   - **科技服务**：服务国内超200家企业客户，涵盖10大行业

### 新事业布局（"2+2"战略）

1. **半导体**：布局半导体制造相关业务
2. **新能源汽车零部件**：切入新能源车供应链
3. **大数据**：元宇宙算力及储能
4. **机器人**：工业机器人及自动化

### 重要客户

- **云服务巨头**：亚马逊AWS、微软Azure、谷歌云等
- **AI芯片巨头**：英伟达（核心合作伙伴）
- **网络设备巨头**：思科、华为等

## 商业模式

### 盈利模式
- **JDM模式**：联合设计制造（Joint Design Manufacture）
  - 与客户联合研发设计
  - 提供从设计到制造的一站式服务
  - 深度绑定客户，提升粘性

- **规模制造**：大规模量产能力
  - 全球化产能布局
  - 自动化产线，降低成本
  - 交付效率高

- **产品升级**：从通用服务器到AI服务器
  - AI服务器单价是传统服务器数倍
  - 毛利率更高
  - 技术门槛更高

### 定价策略
- 成本加成定价
- 规模效应带来的成本优势
- 高端产品（AI服务器）溢价能力更强

### 供应链特点
- **垂直整合**：鸿海精密体系内协同
  - 上游零部件供应
  - 中游设计制造
  - 下游客户资源

- **全球化布局**：
  - 中国大陆：深圳、郑州、成都等
  - 海外：东南亚、墨西哥等
  - 贴近客户，降低风险

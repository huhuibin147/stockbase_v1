# 寒武纪 - 投资笔记

## 投资逻辑

### 核心投资逻辑

**国产AI芯片龙头，商业化落地拐点已至**

1. **国产替代确定性趋势**：
   - 2025年英伟达在国内AI加速卡市场份额从绝对主导降至55%
   - 国产芯片整体份额跃升至41%
   - 寒武纪作为头部国产供应商，直接受益于这一历史性窗口

2. **业绩爆发式增长**：
   - 2025年营收64.97亿，同比+453%
   - 2025年净利润20.59亿，首次年度盈利
   - 机构预测2026年净利润48.4亿、2027年77.9亿

3. **产品线完善，技术领先**：
   - 思元590：性能对标英伟达A100（80%-90%），2025年批量应用
   - 思元690：对标英伟达H100，2026年小批量交付
   - 国内首家实现"云边端"全场景覆盖的AI芯片厂商

4. **生态壁垒深厚**：
   - 自研MLU架构与Cambricon指令集
   - 累计专利超1478项
   - NeuWare软件栈持续迭代，适配主流大模型与框架

### 成长驱动因素

1. **智算中心建设加速**：
   - 各地政府主导的智算中心项目大规模建设
   - 国产算力芯片需求持续爆发

2. **大模型训练与推理需求**：
   - AI大模型迭代推动算力需求指数级增长
   - 思元590/690在训练和推理场景均有优势

3. **思元690放量**：
   - 2026年战略级产品，对标H100
   - 性能较思元590提升70%
   - 小批量交付后逐步放量

4. **软件生态完善**：
   - NeuWare软件栈持续迭代
   - 适配所有国内主流大模型
   - 生态壁垒越筑越高

## 潜在风险

### 核心风险

1. **技术风险**：
   - 思元690性能不及预期或交付延迟
   - 与英伟达旗舰产品仍有代际差距
   - 先进制程受限（台积电7nm可能受美国制裁影响）

2. **市场风险**：
   - 华为昇腾竞争加剧（华为出货量81.2万张，远超寒武纪）
   - 阿里平头哥等互联网大厂自研芯片
   - 沐曦股份、摩尔线程等GPU公司竞争

3. **财务风险**：
   - 经营现金流持续为负
   - 存货跌价风险（存货+178%）
   - 高估值回调风险（PE 267倍）

4. **政策风险**：
   - 美国制裁升级，限制台积电代工
   - HBM显存等关键材料供应受限
   - 国产替代政策力度不及预期

5. **生态风险**：
   - 软件生态不及华为昇腾完善
   - 开发者社区活跃度不足
   - 大模型厂商适配成本高

### 风险对冲建议

1. **分散投资**：不要单一重仓寒武纪，可以配置国产AI芯片组合（寒武纪+海光信息）
2. **分批建仓**：避免一次性重仓，逢低分批买入
3. **止损设置**：如果股价跌破800元（市值3300亿），考虑止损
4. **持续跟踪**：密切关注思元690交付进度、客户反馈、财务数据

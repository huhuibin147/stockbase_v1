## 主营业务

### 核心产品线

1. **数据中心业务**（收入占比约90%）
   - **Hopper架构**：H100、H200（2022-2025年主力产品）
     - H100：台积电4nm工艺，800亿晶体管，HBM3显存，Transformer引擎
     - H200：H100升级版，HBM3e显存，容量141GB，带宽4.8TB/s
   - **Blackwell架构**：B200、GB200、B300（2025-2026年主力产品）
     - B200：2080亿晶体管，FP4精度，推理吞吐量较H100提升15倍
     - GB200：双B200芯片+Grace CPU，支持NVLink互联
     - B300：Blackwell升级版，2026年出货主力
   - **Rubin架构**：下一代产品（预计2026年下半年出货）
   - **网络硬件**：InfiniBand、以太网交换机、NVLink互联技术

2. **游戏和AI PC业务**（收入占比约7%）
   - GeForce RTX系列游戏显卡（RTX 4090/5090等）
   - AI PC相关GPU
   - 2026财年Q1创历史记录：38亿美元

3. **专业可视化业务**（收入占比约1%）
   - RTX专业工作站GPU
   - 用于设计、影视制作、科学可视化
   - 2026财年全年收入32亿美元（+70%）

4. **汽车和机器人业务**（收入占比约1%）
   - DRIVE Orin/Thor自动驾驶芯片
   - Isaac机器人平台
   - 2026财年全年收入23亿美元（+39%）

### 软件生态

- **CUDA平台**：GPU并行计算平台，开发者超500万
- **AI软件栈**：TensorRT、cuDNN、NCCL等
- **企业软件**：NVIDIA AI Enterprise、Omniverse
- **云服务**：DGX Cloud、NVIDIA Base Command

### 重要客户

- **云厂商**：微软Azure、亚马逊AWS、谷歌云、阿里云、腾讯云

- **企业客户**：宝马、沃尔玛、摩根大通等
- **政府/科研**：各国超算中心、研究机构

## 商业模式

### 盈利模式
- **GPU芯片销售**：数据中心GPU、游戏GPU直接销售
- **系统解决方案**：DGX服务器、HGX模组
- **软件授权**：CUDA生态、企业软件订阅
- **云服务**：DGX Cloud云服务

### 定价策略
- **高端定价**：H100单卡售价约2.5-4万美元，B200更高
- **生态锁定**：CUDA生态形成强大壁垒，客户迁移成本高
- **全栈销售**：芯片+网络+软件+系统整体解决方案

### 供应链特点
- **Fabless模式**：专注于芯片设计，委托台积电制造
- **先进制程**：采用台积电4nm/3nm工艺
- **HBM显存**：依赖三星、SK海力士、美光
- **封装测试**：日月光、台积电CoWoS封装

# 佰维存储 - 投资笔记

## 核心投资逻辑

### 1. AI端侧存储全球领先

**AI端侧爆发**：
- AI眼镜、AR、智能手表等AI端侧设备快速普及
- 佰维是Meta AI眼镜ePOP模组独家供应商
- AI相关业务营收同比增长294%，2025年收入17.51亿元
- ePOP全球市占率第一，业务增速超500%

**技术壁垒**：
- 全球唯一具备晶圆级先进封测能力的独立存储解决方案商
- ePOP厚度仅0.6mm，存储与内存二合一集成
- Mini SSD入选《时代》周刊"年度最佳发明"

### 2. 存储超级周期反转

**周期判断**：
- 存储产业上一轮周期高峰在2021年
- 2022年起经历六个季度深度下跌
- 本轮上行周期起始于2023年三季度，景气度预计持续至2027年
- 供需格局：AI服务器虹吸产能，供给刚性，需求爆发

**价格走势**：
- 2025年存储价格持续上行
- 2025年毛利率21.44%，较2024年提升7.53pct
- 机构预测2026年全球DRAM增长51%，NAND增长45%

### 3. 全链条能力构建壁垒

**五大核心能力**：
1. **存储介质特性研究**：深入理解NAND/DRAM特性
2. **自研主控芯片**：SP1800已量产，UFS投片
3. **固件算法开发**：多年技术积累
4. **先进封测制造**：全球唯一晶圆级先进封测
5. **全球品牌运营**：佰维+国际品牌授权

### 4. 国产替代加速

**国产化空间**：
- 2025年一季度国产DRAM份额不足5%
- 国产NAND Flash市场份额不足10%
- 政策+市场双重驱动，国产替代进入规模化阶段

**国家级认可**：
- 科创50成分股
- 国家大基金二期战略投资
- 中国互联网投资基金、中科院旗下基金战略投资
- 国家级专精特新"小巨人"

## 财务亮点

### 业绩爆发
- 2025年营收113.02亿（+68.82%）
- 归母净利润8.53亿（+429.07%）
- 毛利率21.44%，创近年新高
- ROE 19.78%，较2024年大幅提升

### 业务结构优化
- 嵌入式存储占比60.9%，是核心基本盘
- AI相关业务17.51亿，同比增长294%
- 企业级存储营收同比增长超300%

### 研发投入
- 2025年研发费用6.32亿（+41.34%）
- 2020-2025年累计研发投入16.20亿
- 研发费用率约5.6%

## 风险提示

### 1. 周期性风险
- 存储行业强周期性，毛利率随价格大幅波动
- 2023年行业下行期曾巨亏6.24亿
- 需持续跟踪存储价格走势

### 2. 供应链风险
- 存储晶圆依赖三星、美光等海外原厂
- 地缘政治风险：出口管制
- 客户集中度高，头部客户贡献显著

### 3. 竞争风险
- 三星、美光等原厂垂直整合模组业务
- 国内同行（江波龙、德明利等）竞争加剧
- 技术迭代持续高压

### 4. 财务风险
- 存货规模大（78亿+）
- 经营现金流为负（-19.65亿）
- 资产负债率64.46%，处于行业较高水平

# 江波龙 - 投资笔记

## 核心投资逻辑

### 1. 存储超级周期反转

**周期判断**：
- 存储产业上一轮周期高峰在2021年
- 2022年起经历六个季度深度下跌
- 本轮上行周期起始于2023年三季度，景气度预计持续至2027年
- 供需格局：AI服务器虹吸产能，供给刚性，需求爆发

**价格走势**：
- 2025年上半年DRAM价格指数上涨47.7%，NAND上涨9.2%
- 存储价格上涨对业绩影响尚处初期阶段
- 机构预测2026年全球DRAM增长51%，NAND增长45%

### 2. AI驱动存储需求爆发

**AI服务器存储需求**：
- DRAM需求是普通服务器的8倍
- NAND需求是普通服务器的3倍
- AI服务器出货量同比增长33%至190万台

**企业级SSD市场**：
- 2025年全球企业级SSD市场达245亿美元（+38.2%）
- 预计2030年突破800亿美元，复合增速超26%
- 国内企业级存储国产替代率不足15%，空间巨大

### 3. 全链条能力构建壁垒

**四大核心能力**：
1. **自研主控芯片**：累计部署超1亿颗，UFS4.1采用三星5nm制程
2. **固件算法**：20年经验，400+人团队
3. **封装测试**：收购力成苏州，良品率99.5%
4. **品牌渠道**：FORESEE行业品牌 + Lexar消费品牌

### 4. 国产替代加速

**国产化空间**：
- 2025年一季度国产DRAM份额不足5%
- 国产NAND Flash市场份额不足10%
- 政策+市场双重驱动，国产替代进入规模化阶段

**长江存储、长鑫存储**：
- 技术持续突破，产能扩张
- 为江波龙等模组厂商提供更多国产晶圆选择

## 财务亮点

### 业绩反转
- 2025年归母净利润12.5-15.5亿（+150%~210%）
- 扣非净利润11.3-13.5亿（+578%~710%）
- Q3单季营收65.39亿，创历史新高
- Q3单季毛利18.92亿元，净利率11%

### 业务结构优化
- 企业级存储前三季度收入12.06亿（+138.66%）
- Lexar品牌前三季度收入35.44亿（+40%）
- 高毛率业务占比持续提升

### 现金流改善
- 2025年Q2以来经营现金流由负转正
- 存储价格上涨带动盈利质量提升

## 风险提示

### 1. 周期性风险
- 存储行业强周期性，毛利率随价格大幅波动
- 2023年行业下行期曾巨亏8.28亿，毛利率跌至0.81%
- 需持续跟踪存储价格走势

### 2. 供应链风险
- 存储晶圆依赖三星、SK海力士、美光等海外原厂
- 地缘政治风险：美国贸易政策变动
- 与长江存储、长鑫存储合作降低风险

### 3. 竞争风险
- 三星、SK海力士等原厂垂直整合模组业务
- 国内同行（佰维存储、德明利等）竞争加剧
- 价格战可能压缩利润空间

### 4. 财务风险
- 存货规模大（85亿+），占资产比超40%
- 存货跌价风险（跌价准备11.52亿）
- 资产负债率52%，处于行业中较高水平

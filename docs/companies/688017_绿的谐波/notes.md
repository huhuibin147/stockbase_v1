# 绿的谐波 - 投资笔记

## 一、投资逻辑

### 1.1 核心优势

| 优势维度 | 具体内容 |
|----------|----------|
| **技术壁垒** | 自主研发"P型齿"谐波减速器，精度≤5弧秒，专利452项（发明专利占61%） |
| **成本优势** | 同规格产品价格为哈默纳科的40-50%，毛利率高于哈默纳科约10pct |
| **产业链卡位** | 全球唯一实现全零部件自主配套的厂商，交货周期仅2个月 |
| **客户绑定** | 深度绑定特斯拉、优必选、宇树科技、埃斯顿等头部企业，人形机器人订单覆盖全球超30家主流整机厂商 |
| **产能领先** | 2025年产能50万台/年，新建项目将再增100万台/年 |

### 1.2 成长驱动

1. **人形机器人爆发**
   - 2025年为量产元年，全球出货量预计达1万台
   - 2030年预计突破51万台（IDC预测）
   - 单机需14-20颗谐波减速器，价值量约2.8-4万元
   - 潜在市场规模超150亿元（2025-2032八年CAGR 83%）
   - **最新订单数据**：2026年Q1千万元以上订单合计12.76亿元，其中人形机器人相关订单5.82亿元，同比暴涨286.43%
   - 3月单月订单4.12亿元，环比增长108.16%
   - 2025年机器人相关订单总额超8亿元，特斯拉相关约占12%-15%

2. **国产替代加速**
   - 工业机器人领域国产渗透率已超40%
   - 人形机器人领域有望达30%以上
   - 绿的谐波国内市占率绝对领先

3. **产能释放**
   - 定增募资20亿元扩产
   - 墨西哥合资工厂规划50万套/年
   - 订单已排至2027年第三季度
   - 2025年底产能80万台，2026年规划扩至150万台，当前产能利用率约95%（满产）
   - 2026年人形机器人专用产能规划40万台，月产10-15万个

### 1.3 行业空间

- 2024年中国减速器市场规模约1447亿元
- 2026年预计达1672亿元
- 全球谐波减速器市场2032年预计达29.27亿美元（CAGR 26.5%）

## 二、竞争格局

### 全球排名

| 排名 | 公司 | 全球市占率 | 特点 |
|------|------|------------|------|
| 1 | 日本哈默纳科 | 38-51% | 技术领先，IH齿形专利，扩产保守 |
| 2 | **绿的谐波** | 15-26% | 国产龙头，性价比高，快速扩产 |
| 3 | 同川科技（汉宇集团） | 约15% | 扩产激进，同比增300%+，正在切入人形机器人供应链 |
| 4 | 来福谐波 | 约5% | 中端市场、SCARA机器人为主，产能60万台/年，技术落后于绿的 |
| 5 | 杉川谐波（大族传动） | 约5%-8% | 半导体/激光设备专用，2024年出货13万套 |

### 竞争态势
- **短期**：哈默纳科技术仍领先，但扩产保守（2026年规划仅347万台/年）
- **中期**：绿的谐波凭借性价比+产能+服务响应快速抢份额，同川科技是最大变数（哈默纳科技术奠基人领衔）
- **长期**：技术迭代和成本控制能力决定胜负，可能出现新路线（如直驱电机）

## 四、潜在风险

### 4.1 高风险

1. **估值回调风险**
   - 当前PE 291倍，任何业绩不及预期都可能导致大幅回调
   - 2026年4月23日年报发布后股价单日跌7.57%，市场分歧加大

2. **人形机器人产业化不及预期**
   - 人形机器人量产进度存在不确定性
   - 若Tesla Optimus等推迟，下游需求将受冲击
   - 2026年4月特斯拉Optimus Gen-3已在上海工厂交付50台，年底产能1000台/月（年产能1.2万台），与百万台长期预期差距巨大

3. **技术路线变更**
   - 若出现替代谐波减速器的新技术（如新型电机直驱），公司面临技术淘汰风险

### 4.2 中等风险

4. **竞争加剧**
   - 国内外谐波减速器企业已超50家
   - 同川科技等追赶者扩产激进（哈默纳科技术奠基人领衔，产品寿命达1万小时逼近绿的）
   - 价格战可能侵蚀利润率
   - 来福谐波技术落后但产能60万台/年，杉川谐波半导体领域市占超20%

5. **客户集中度**
   - 深度绑定特斯拉等大客户，存在大客户依赖风险
   - 若特斯拉自研或切换供应商，影响巨大

6. **产能消化风险**
   - 大规模扩产后若需求不及预期，产能利用率下降
   - 固定资产折旧将拖累利润

### 4.3 低风险

7. **财务风险**：负债率仅9.82%，财务极其稳健
8. **管理层风险**：创始团队稳定，技术背景深厚

## 六、投资观点

### 短期（6个月）
- 估值偏高，业绩兑现压力大
- 股价波动可能加剧，需警惕回调风险
- 建议观望或轻仓参与

### 中期（1-2年）
- 人形机器人量产+产能释放，业绩有望持续高增
- 若PE回归至100-150倍区间，具备配置价值
- 核心变量：人形机器人出货量

### 长期（3-5年）
- 若人形机器人成为万亿级市场，绿的谐波作为核心零部件龙头将充分受益
- 全球第二→挑战第一的逻辑有机会兑现
- 关注技术迭代和竞争格局变化

### 总体评价
> 绿的谐波是**人形机器人产业链中最纯正的减速器标的**，技术、成本、产能、客户全方位领先。但**当前估值已透支未来2-3年增长**，适合**长期看好机器人产业、能承受高波动的投资者**。短期建议等待更好的买入时机（如估值回调至PE 150倍以下或业绩超预期验证）。

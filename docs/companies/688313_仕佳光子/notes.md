# 仕佳光子 (688313) 投资笔记

## 投资逻辑

### 核心投资逻辑

#### 1. 行业赛道：AI算力爆发，光芯片需求激增
- **AI算力需求指数级增长**：英伟达GPU供不应求，数据中心建设加速
- **光模块升级**：从400G向800G/1.6T演进，光芯片需求翻倍
- **供需缺口**：高端光芯片供需缺口20%-30%，处于卖方市场
- **市场规模**：2025年全球光芯片市场128亿美元，2025-2028年CAGR达18.2%

#### 2. 公司定位：国内光芯片IDM龙头
- **技术壁垒**：国内少数具备"无源+有源"双芯片平台、全流程IDM能力
- **市场地位**：PLC分路器芯片全球市占率超50%，AWG芯片国内最大供应商
- **国产替代**：国内唯一实现1.6T AWG芯片量产的厂商
- **客户资源**：英伟达、Intel、中际旭创、华为等头部客户

#### 3. 业务结构：无源+有源双轮驱动
- **无源芯片**：PLC分路器芯片全球第一，现金牛业务
- **有源芯片**：DFB、EML芯片快速突破，成长引擎
- **协同发展**：三大业务板块协同，抗风险能力强

#### 4. 技术突破：1.6T产品量产，CPO布局前瞻
- **1.6T AWG芯片**：国内唯一量产，已通过Intel认证，切入英伟达GB200供应链
- **硅光芯片**：高功率CW DFB激光器（500mW-1000mW）实现小批量出货
- **CPO布局**：提前布局下一代技术，抢占未来制高点

## 投资思考

### 核心问题

#### 1. 与源杰科技相比，仕佳光子的竞争优势是什么？
**仕佳光子优势**:
- PLC分路器芯片全球市占率超50%，绝对龙头
- 无源芯片技术领先，成本优势显著
- 业务多元化，抗风险能力强
- 客户资源丰富，覆盖面广

**源杰科技优势**:
- 有源芯片技术领先，100G EML已量产
- 数据中心业务占比更高（65.4%）
- 盈利能力更强（毛利率58%）
- 客户集中度虽高，但绑定头部客户

**结论**: 
- 仕佳光子在无源芯片领域优势明显，现金牛业务稳定
- 源杰科技在有源芯片领域领先，成长性更强
- 两者定位不同，仕佳光子更稳健，源杰科技更激进

#### 2. 应收账款和存货风险如何化解？
**应收账款**:
- 主要来自优质客户，信用风险较低
- 应收账款周转率提升，回款加快
- 需持续关注回款情况

**存货**:
- 主要因订单增长备货，属正常经营行为
- 客户订单年均增长20%，长期收益可覆盖短期占用
- 需关注存货跌价风险

**应对**:
- 加强应收账款和存货管理
- 提高周转效率
- 优化客户结构

#### 3. 估值是否合理？
**当前估值**:
- 静态PE约110倍，显著高于行业平均
- 动态PE（2026年）约55-62倍，仍偏高
- 估值已透支未来2-3年增长预期

**估值支撑**:
- AI算力需求爆发，行业高景气
- 公司技术领先，成长确定性强
- 国产替代空间广阔

**估值风险**:
- 业绩增速可能放缓
- 市场预期过高
- 估值回调压力大

**结论**: 
- 估值偏高，需等待回调机会
- 关注业绩兑现情况
- 理性看待估值，设置安全边际

#### 4. 业绩增速能否持续？
**2025年高增长原因**:
- AI算力需求爆发
- 低基数效应
- 产品结构优化

**2026年增速预期**:
- 机构预测增速52%-93%
- 显著低于2025年的426%
- 低基数效应消退

**可持续性分析**:
- AI算力需求持续，行业景气度高
- 1.6T AWG芯片大规模量产
- 有源芯片逐步量产
- 国产替代加速

**结论**: 
- 业绩仍将保持高增长，但增速可能放缓
- 需关注订单持续性和行业景气度
- 理性看待增速变化

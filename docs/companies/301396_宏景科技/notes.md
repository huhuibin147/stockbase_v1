# 宏景科技 - 投资笔记

## 投资逻辑

### 核心投资逻辑

**从传统智慧城市向AI算力服务转型，业绩爆发式增长，但估值极高**

1. **算力业务爆发**：
   - 2025年前3季度营收15.51亿（+595%），净利润1.08亿（扭亏为盈）
   - 算力业务是真正将公司体量拉起来的核心业务
   - 智算中心建设需求爆发，订单大幅增长

2. **战略转型**：
   - 从传统智慧城市企业向AI算力服务商转型
   - 与华为、浪潮、腾讯等大厂合作
   - 设立合资公司深圳宏景纵横，加码算力业务

3. **AI算力赛道**：
   - AI大模型训练和推理需求爆发
   - 智算中心建设加速
   - 算力服务市场空间广阔

4. **业绩拐点**：
   - 2024年亏损0.76亿，2025年扭亏为盈
   - 从传统低增长业务转向高增长算力业务

### 成长驱动因素

1. **智算中心建设**：
   - 各地政府和企业加速建设智算中心
   - 算力基础设施需求持续爆发

2. **与大厂合作**：
   - 与华为、浪潮、腾讯等合作，获取算力资源和技术支持
   - 提升订单获取能力

3. **区域扩张**：
   - 总部广州，在全国多地设立分支机构
   - 从区域龙头向全国扩张

4. **政策驱动**：
   - 国家推动AI算力基础设施建设
   - "东数西算"工程等政策支持

## 潜在风险

### 核心风险

1. **估值过高**：
   - PE 769倍，远高于行业中位107倍
   - 风险：如果业绩增速不及预期，可能面临估值大幅回调（-50%以上）

2. **资产负债率快速上升**：
   - 从58.5%升至72.4%
   - 风险：财务杠杆过高，偿债压力增大

3. **现金流风险**：
   - 经营现金流6139万，远小于净利润1.08亿
   - 投资现金流-8.3亿，依赖融资7.22亿
   - 风险：如果融资受阻，可能影响扩张

4. **算力业务可持续性**：
   - 算力业务是2025年增长核心驱动
   - 风险：如果算力需求放缓或竞争加剧，可能影响增长

5. **转型风险**：
   - 从传统智慧城市向算力服务转型
   - 风险：转型能否成功存在不确定性

6. **竞争风险**：
   - 算力服务市场竞争加剧
   - 华为、浪潮等大厂直接竞争
   - 风险：市场份额和毛利率可能下降

7. **应收账款风险**：
   - 智慧城市项目回款周期长
   - 风险：可能存在坏账风险

### 风险对冲建议

1. **小仓位试探**：当前估值极高，建议小仓位（5-10%）试探
2. **分批建仓**：避免一次性重仓，逢低分批买入
3. **止损设置**：如果股价跌破150元（市值230亿），考虑止损
4. **持续跟踪**：密切关注订单、毛利率、现金流、资产负债率

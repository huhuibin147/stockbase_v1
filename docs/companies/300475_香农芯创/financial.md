## 二、盈利能力分析

### 2.1 毛利率

| 业务 | 2025年毛利率 | 同比变化 | 说明 |
|------|--------------|----------|------|
| 电子元器件分销 | 2.86% | -1.13pct | 分销行业竞争激烈，盈利空间薄 |
| 集成电路（含存储器） | 2.86% | - | 同上 |
| 整体 | 极低 | - | 分销模式决定低毛利 |

**分析**：
- 分销业务毛利率仅2.86%，反映电子元器件分销行业竞争激烈
- 盈利空间相对较薄，主要靠规模效应
- 自研存储（海普存储）毛利率较高，但占比仅4.70%

### 2.2 业务板块营收结构（2025年）

| 业务板块 | 营收 | 占比 | 同比增速 |
|----------|------|------|----------|
| 电子元器件分销 | 332.25亿 | 94.25% | +40.90% |
| 海普存储（自研） | 16.58亿 | 4.70% | +467.87% |
| 减速器 | 3.67亿 | ~1% | -7.90% |

**分析**：
- 分销业务是绝对基本盘，占比超94%
- 海普存储爆发式增长，同比激增近5倍
- 2025年Q4海普存储单季收入13亿元

### 2.3 费用率

| 费用 | 2025年 | 同比增速 | 说明 |
|------|--------|----------|------|
| 销售费用 | 9005.58万 | +36.69% | 主要为奖金增加 |
| 管理费用 | 2.09亿 | +6.36% | - |
| 财务费用 | 2.38亿 | -5.71% | - |

**分析**：
- 销售费用增长较快，主要为奖金增加
- 管理费用、财务费用相对稳定

## 七、风险提示

### 7.1 供应商依赖风险
- 向第一大供应商SK海力士采购额占比超50%
- 代理权一年一签，存在续约风险

### 7.2 存货跌价风险
- 存储芯片价格周期性极强
- 若价格反转，存货将面临大幅减值风险

### 7.3 商誉减值风险
- 2021年收购联合创泰形成10.90亿商誉
- 如果半导体行业出现趋势性下降，存在商誉减值风险

### 7.4 业绩透支风险
- 2026年Q1暴增的业绩可能源于低价库存的集中出货
- 后续若备货不足，高增长难以为继

### 7.5 估值风险
- 当前估值处于历史绝对高位
- 任何业绩或行业景气度不及预期都可能引发戴维斯双杀

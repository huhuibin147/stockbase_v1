# 香农芯创 - 投资笔记

## 核心投资逻辑

### 1. SK海力士HBM大陆独家代理权

**稀缺性**：
- SK海力士HBM在中国大陆云服务领域最大的授权代理商
- 在AI服务器爆发周期中，HBM是必备组件，供需极度紧张
- 连接全球存储巨头与国内AI算力需求的唯一通道
- 牌照壁垒短期内无法被复制

**客户绑定**：
- 直接绑定阿里、腾讯、字节、华为等AI算力与云服务巨头
- 下游覆盖阿里巴巴、腾讯、字节跳动、百度等一线互联网厂商
- 华勤通讯等大型ODM企业

### 2. 存储超级周期反转

**周期判断**：
- 2025年三季度起全球存储市场迎来颠覆性涨价行情
- 大型语言模型持续迭代，AI数据中心大规模建设
- 服务器对DRAM和NAND闪存需求呈几何级增长
- 行业机构普遍对存储芯片景气度保持乐观预期

**业绩爆发**：
- 2026年Q1归母净利润11.40-14.80亿元，同比暴增6714%-8747%
- 暂列A股一季度净利"预增王"
- 仅一个季度利润超过2025年全年的两倍

### 3. "分销+产品"双轮驱动

**分销业务（基本盘）**：
- 2025年营收332.25亿（+40.90%），占比94.25%
- 代理SK海力士、AMD、联发科、兆易创新等一线品牌
- 提供稳定现金流

**自研存储（第二增长曲线）**：
- 海普存储2025年营收16.58亿（+467.87%）
- 2025年Q4单季收入13亿，加速放量
- 首次实现年度规模盈利
- 企业级DDR4、DDR5、Gen4 eSSD已量产

### 4. 存力+算力布局

**算力延伸**：
- 2025年8月拟出资3000万元设立无锡新威智算科技
- 打造区域性低成本、绿色化数字基础设施
- 从存储领域向算力基础设施方向战略延伸
- 形成"存力+算力"双轮驱动

### 5. 国产替代

**海普存储定位**：
- 深度服务国家大数据产业
- 围绕国产化、定制化路线
- 服务于"东数西算"智算中心
- 与国内软件厂商合作提供全闪存储解决方案

## 财务亮点

### 业绩爆发
- 2025年营收352.51亿（+45.24%）
- 2025年归母净利润5.45亿（+106.23%）

- 境内收入51.81亿（+267.01%），占比从5.82%提升至14.70%
- 自研存储占比持续提升

- 经营现金流由负转正，达14.38亿
- 消化库存速度加快

## 风险提示

### 1. 供应商依赖风险
- 向第一大供应商SK海力士采购额占比超50%
- 代理权一年一签，存在续约风险
- 如果失去代理权，业务将受到重大打击

### 2. 分销毛利率极低
- 分销业务毛利率仅2.86%
- 盈利空间薄，主要靠规模效应
- 行业竞争激烈

### 3. 存货跌价风险
- 存储芯片价格周期性极强
- 2025Q3末存货18.13亿
- 若价格反转，将面临大幅减值风险

### 4. 商誉减值风险
- 2021年收购联合创泰形成10.90亿商誉
- 占总资产9.83%
- 如果半导体行业出现趋势性下降，存在商誉减值风险

### 5. 业绩透支风险
- 2026年Q1暴增的业绩可能源于低价库存的集中出货
- 后续若备货不足，高增长难以为继

### 6. 估值风险
- 当前估值处于历史绝对高位
- PE(TTM)约280.60倍，远超行业平均134.06倍
- PB(MRQ)约21.43倍，远超行业平均6.55倍
- 任何业绩或行业景气度不及预期都可能引发戴维斯双杀
- 合理市值区间测算：217亿-260.4亿元，当前市值727亿，高估约179%

### 7. 高管减持信号
- 2025年12月-2026年1月，董事总经理李小红减持19.95万股
- 财务总监苏泽晶减持2.92万股
- 业绩预告前精准减持，传递内部人士对短期股价高位的谨慎信号

# 海光信息 - 投资笔记

## 核心投资逻辑

### 1. 国产算力替代的确定性机会

**政策驱动**：
- 2027年核心系统100%国产化目标
- 当前高端服务器CPU国产化率不足10%，替代空间巨大
- 信创产业从党政向金融、电信、能源等行业深水区延伸

**市场驱动**：
- AI算力需求爆发式增长，年复合增长率超50%
- 海外高端芯片供应受限，国产替代需求迫切
- 海光信息作为"卖铲人"，无论下游大模型谁胜出，算力需求都是确定性增量

### 2. "CPU+DCU"双轮驱动

**CPU业务（压舱石）**：
- x86生态兼容性壁垒，客户迁移成本极低
- 金融、电信等核心行业突破（工商银行30亿订单）
- 海光四号放量，海光五号预计2026年发布

**DCU业务（增长极）**：
- 深算三号量产，深算四号研发顺利
- 兼容类CUDA生态，降低客户迁移门槛
- 2025年万卡级集群部署，300+应用场景落地
- DCU营收占比超50%，在手订单超120亿

### 3. 生态壁垒

- "光合组织"6000+生态合作伙伴
- 15000+项软硬件测试
- 适配365款主流大模型，覆盖全球99%非闭源大模型
- 开放HSL总线互联协议，构建算力生态闭环

## 财务亮点

### 成长性
- 2021-2025年营收复合增长率超50%
- 2025年营收143.77亿，同比+56.92%
- 2026年Q1营收40.34亿，同比+68.06%
- 合同负债20.19亿，同比+123.42%，订单饱满

### 盈利能力
- 毛利率57.83%，虽有所下降但仍处高位
- 净利率25.17%，远超普通硬件厂商
- ROE 11.87%，同比+1.95个百分点

### 现金流
- 经营现金流20.97亿，同比+114.61%
- 货币资金88.74亿+定期存款70亿，现金储备充足
- 资产负债率仅27.13%，财务安全垫厚实

## 风险提示

### 1. 供应链风险
- 先进制程代工依赖台积电
- EDA工具及核心IP部分依赖海外
- 美国出口管制升级可能影响研发与量产

### 2. 竞争风险
- CPU赛道：华为鲲鹏、飞腾等ARM架构厂商持续发力
- AI芯片赛道：华为昇腾已形成生态与份额绝对领先
- 寒武纪、壁仞等企业加速迭代
- 英伟达特供芯片持续降价，可能挤压毛利率

### 3. 财务风险
- 应收账款增速高于营收增速（77.29% vs 56.92%）
- 短期债务占比高（88.6%）
- 毛利率持续下降趋势（63.72%→57.83%）
- 客户集中度高（前五大客户占比90.28%）

### 4. 技术风险
- 高端DCU算力与能效比仍落后于英伟达H100
- 技术迭代不及预期
- 高研发投入压制短期利润（研发费用率31.78%）

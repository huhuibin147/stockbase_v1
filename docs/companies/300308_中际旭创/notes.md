# 中际旭创 (300308) 投资笔记

## 投资评级

**评级**: 买入
**目标价**: 980-1050元
**当前价**: 857.03元（2026-04-27）
**上涨空间**: +14.4%~+22.5%

## 核心投资逻辑

### 1. 全球光模块绝对龙头，市占率持续领先

#### 市场地位
- 2025年全球市占率23.4%，连续三年蝉联第一
- 800G光模块市占率超40%，全球第一
- 1.6T光模块市占率50%-70%，形成垄断优势
- 全球唯一同时具备800G、1.6T大规模量产交付能力的厂商

#### 竞争优势
- 技术领先：硅光芯片良率95%，技术领先行业6-12个月
- 规模优势：2025年化产能2800多万只，规模效应明显
- 客户优势：深度绑定英伟达、谷歌、微软、Meta等AI巨头

### 2. 深度绑定英伟达，订单能见度极高

#### 英伟达合作
- 中际旭创是英伟达1.6T硅光模块的独家供应商
- 占英伟达1.6T总采购量约80%
- 2025-2027年三年长协，订单已排至2027年Q2
- 来自英伟达的营收占比约60%

#### 订单能见度
- 订单排至2027年Q2
- 部分客户规划至2028年
- 续单率高达97%
- 在手订单超300亿元

### 3. AI算力爆发，行业景气度持续高企

#### 行业趋势
- 全球AI大模型迭代提速，算力基础设施投入加大
- 2026年全球AI光模块市场规模260亿美元（+57%）
- 800G、1.6T需求爆发式增长
- 数据中心向CPO+硅光演进

#### 市场空间
- 2026年800G光模块需求量3400万只（+89%）
- 2026年1.6T光模块需求量1400万只（+180%）
- 2029年全球数通光模块市场将达258亿美元

### 4. 业绩爆发式增长，盈利能力卓越

#### 业绩表现
- 2025年营收382.4亿元（+60.25%）
- 2025年净利润107.97亿元（+108.78%）
- 2026Q1净利润57.35亿元（+262%），单季已超2024年全年
- 连续12季度净利润环比增长

#### 盈利能力
- 毛利率从2024年33.8%提升至2026Q1的46.06%
- 净利率达32.40%
- ROE达43.84%，A股顶尖水平
- 经营现金流/净利润 > 100%

### 5. 技术护城河深厚，持续研发投入

#### 技术优势
- 自研硅光芯片，良率达95%
- 持有137项CPO核心专利
- 全球首发12.8T XPO光模块
- 技术领先行业6-12个月

#### 研发投入
- 2025年研发费用16.76亿元（+25.77%）
- 研发人员2169人
- 拥有专利411件

## 主要风险

### 1. 存货风险（中等风险）

#### 风险描述
- 存货规模大（156.72亿元），占净资产44%
- 存货周转率下降19.31%
- 光模块技术迭代快，存在跌价风险

#### 应对措施
- 订单能见度高，存货有订单支撑
- 已计提跌价准备2.98亿元
- 加强存货管理

### 2. 客户集中度风险（高风险）

#### 风险描述
- 前五大客户占比75.98%
- 英伟达占比约60%
- 客户依赖度极高

#### 应对措施
- 与英伟达签订长期协议
- 续单率高达97%
- 拓展其他客户

### 3. 供应链风险（高风险）

#### 风险描述
- 高端光芯片90%依赖进口
- 前五大供应商采购占比51.5%
- 存在"卡脖子"风险

#### 应对措施
- 加大采购力度，与供应商签署保障协议
- 导入新的供应商
- 预付款项大幅增长（14.88亿元，增长超10倍）

### 4. 汇率风险（中等风险）

#### 风险描述
- 境外收入占比90.58%
- 2025年汇兑损失3.18亿元
- 2026Q1财务费用2.51亿元（+1550%）

#### 应对措施
- 加强汇率风险管理
- 使用金融工具对冲

### 5. 地缘政治风险（高风险）

#### 风险描述
- 主要客户在北美
- 贸易摩擦风险
- 出口管制风险

#### 应对措施
- 泰国生产基地规模化制造
- 优化全球产能配置

## 投资思考

### 为什么看好中际旭创？

#### 1. 行业赛道好
- AI算力爆发，光模块是核心受益环节
- 行业景气度持续高企，2026年市场规模+57%
- 技术迭代快，龙头优势明显

#### 2. 公司地位强
- 全球光模块绝对龙头，市占率23.4%
- 800G市占率40%+，1.6T市占率50%-70%
- 技术领先，规模领先，客户领先

#### 3. 业绩确定性高
- 深度绑定英伟达，订单排至2027年
- 连续12季度净利润环比增长
- 2026Q1净利润+262%，单季已超2024年全年

#### 4. 盈利能力强
- ROE 43.84%，A股顶尖水平
- 毛利率46.06%，净利率32.40%
- 经营现金流/净利润 > 100%

### 主要担忧是什么？

#### 1. 客户集中度太高
- 英伟达占比60%，一旦英伟达削减采购，业绩将断崖式下跌
- 需要持续关注英伟达的资本开支和技术路线

#### 2. 供应链"卡脖子"
- 高端光芯片90%依赖进口
- 地缘政治风险可能影响供应链安全

#### 3. 存货风险
- 存货规模大，技术迭代快
- 一旦需求放缓，存货跌价风险大

#### 4. 估值不便宜
- 当前PE 41.61倍，虽然考虑成长性后估值合理，但仍有回调风险

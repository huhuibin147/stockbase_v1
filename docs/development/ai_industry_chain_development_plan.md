- React 18: 用户界面构建

- Ant Design: UI组件库
- React Router: 路由管理

### 后端技术栈
- Node.js: 运行环境
- Express: Web框架
- PostgreSQL: 关系型数据存储
- Neo4j: 图数据库(存储实体关系)
- Elasticsearch: 全文搜索引擎
- Redis: 缓存与会话管理

### 部署与运维
- Docker: 容器化部署
- Nginx: 反向代理
- PM2: Node.js进程管理
- GitHub Actions: CI/CD

## 数据模型设计

### 核心实体
1. **公司(Company)**
   - 基本信息: 名称、代码、上市地、市值
   - 产业链位置: 层级、子行业
   - 业务描述: 主营业务、核心产品

2. **概念(Concept)**
   - 技术术语: HBM、CoWoS、CUDA等
   - 行业趋势: AI Agent、大模型等
   - 出现频率统计

3. **人物(Person)**
   - 基本信息: 姓名、职位
   - 所属公司关联
   - 行业影响力

4. **事件(Event)**
   - 时间信息: 发生日期
   - 事件类型: 发布会、并购、IPO等
   - 相关实体关联

### 关系类型
- 供应链关系: 供应商-客户
- 技术依赖: 技术栈依赖
- 投资关系: 投资-被投资
- 竞争关系: 同行业竞争
- 合作关系: 战略合作

# AI产业链知识库开发文档

## 文档说明

本目录包含AI产业链知识库项目的开发文档，用于记录开发过程、跟踪项目进度。

### 文档结构

1. **ai_industry_chain_development_plan.md**
   - 项目整体开发计划
   - 技术选型方案
   - 数据模型设计
   - 里程碑规划

2. **progress_tracking.md**
   - 每日开发进度记录
   - 功能模块完成情况
   - 问题与解决方案
   - 风险与变更管理

### 使用说明

#### 开发人员
1. 每日开发前，更新`progress_tracking.md`中的计划
2. 开发过程中，及时记录遇到的问题
3. 每日开发结束，更新完成情况
4. 重要决策记录在相应文档中

#### 项目管理
1. 每周review进度跟踪文档
2. 每月更新开发计划
3. 里程碑达成时更新状态
4. 风险发生时及时记录应对措施

### 参考站点
- 目标站点: https://aichainmap.com/home
- 核心特点:
  - 五层产业链框架
  - 多维度索引系统
  - 跨页引用关系
  - 可视化图谱展示

### 开发环境
- 前端: React 18 + Vite + TypeScript
- 后端: Node.js + Express + TypeScript
- 数据库: PostgreSQL + Neo4j
- 搜索引擎: Elasticsearch

### 快速开始

1. 查看开发计划
```bash
cat ai_industry_chain_development_plan.md
```

2. 更新进度
```bash
# 编辑进度跟踪文档
vim progress_tracking.md
```

3. 记录问题
```bash
# 在progress_tracking.md中添加问题记录
```

### 注意事项

1. **及时更新**: 每日开发结束后更新进度
2. **详细记录**: 问题与解决方案要详细记录
3. **定期备份**: 重要文档定期备份
4. **版本控制**: 使用Git管理文档变更

# 财务分析提示词模板

## 使用说明

将以下提示词复制到大语言模型中，配合财务数据进行分析。

---

## 提示词模板

```
你是一位专业的财务分析师，请对以下公司的财务数据进行分析：

### 公司基本信息
- 公司名称：[公司名称]
- 股票代码：[股票代码]
- 所属行业：[行业]
- 分析期间：[报告期]

### 财务数据
[粘贴财务报表数据]

### 分析要求

请从以下几个维度进行分析：

1. **盈利能力分析**
   - 毛利率、净利率变化趋势
   - ROE、ROA水平及变化
   - 核心利润来源

2. **成长性分析**
   - 营收增长率
   - 净利润增长率
   - 增长的可持续性

3. **财务健康度**
   - 资产负债率
   - 流动比率、速动比率
   - 现金流状况

4. **运营效率**
   - 存货周转率
   - 应收账款周转率
   - 总资产周转率

5. **风险提示**
   - 潜在风险点
   - 需要关注的指标

请给出详细的分析结论和投资建议。
```

---

## 使用技巧

1. **数据准备**: 从财报中提取关键财务数据，包括利润表、资产负债表、现金流量表
2. **对比分析**: 可以加入同行业公司数据进行对比
3. **历史对比**: 提供多个报告期的数据，分析趋势变化
4. **关注异常**: 特别标注异常波动的指标，让AI重点分析

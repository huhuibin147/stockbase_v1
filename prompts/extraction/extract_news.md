# 信息提取提示词模板

## 使用说明

将以下提示词复制到大语言模型中，从新闻资讯中提取结构化信息。

---

## 提示词模板

```
你是一位专业的财经信息分析师，请从以下新闻资讯中提取关键信息：

### 原始资讯
标题：[资讯标题]
来源：[媒体名称]
URL：[原文链接]
时间：[发布时间]
内容：
[粘贴新闻全文]

### 提取要求

请按照以下格式提取信息：

1. **基本信息**
   - 标题：
   - 来源：
   - URL：[原文链接，必填]
   - 发布时间：
   - 信息可靠性：[官方公告/主流媒体/行业媒体/一般媒体/市场传闻]

2. **相关主体**
   - 涉及公司：[股票代码+公司名称]
   - 所属行业：
   - 相关人物：

3. **核心内容摘要**（100-200字）
   [客观概括新闻核心内容]

4. **关键数据**
   - 财务数据：[营收、利润等]
   - 经营数据：[销量、产能等]
   - 其他重要数据：

5. **事件影响**
   - 对公司的影响：[正面/负面/中性，具体分析]
   - 对行业的影响：
   - 投资关注点：

6. **标签分类**
   - 公司标签：[股票代码_公司名称]
   - 行业标签：#[行业名称]
   - 主题标签：#[事件类型]

7. **信息来源验证**
   - 是否为官方信息：[是/否]
   - 是否可溯源：[URL是否有效]
   - 需要进一步核实的内容：

请确保提取的信息准确、客观，URL必须保留原文链接。
```

---

## 批量提取模板

```
请从以下多条资讯中批量提取关键信息，每条资讯按以下格式输出：

### 资讯1
标题：xxx
URL：xxx
摘要：xxx
标签：#xxx #xxx

### 资讯2
...

[粘贴多条资讯内容]
```

---

## 使用技巧

1. **保留原文**: 提取前先保存原文，避免链接失效
2. **验证URL**: 确保链接可访问
3. **标注来源**: 区分官方信息和媒体报道
4. **时效性**: 记录信息发布时间
5. **交叉验证**: 重要信息需多方验证
